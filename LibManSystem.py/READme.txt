1. Add a new book

Enter the name, author and quantity of the book you wish to add to the system.

2. Borrow a book

Enter the name of the book which you want to lend out.

3. Return a book

Enter the name of the book which you wish to return to the library.

4. Search for a book

Enter the name of the book you wish to search for.

5. Display all books

Choose this option to display all the books in the library and their quantity.

6. Add a new user

Enter the name and the custom ID number of the user you wish to add to the system.

7. View user details

Enter the User ID of the user you which to search for and display.

8. Display all users

Choose this option to display all the users saved in the library system.

9. Add a new author

Enter the name of the author you wish to add to the library system.

10. View author details

Enter the name of the author you wish to view.

11. Display all authors

Choose this option to display all the authors saved in the system.

12. Quit

Choose this option to quit the system

Thank you for using the Library Management System
